Technology & Innovation

Preparing our Students for a digital future

Digital & Tech Literacy

At Ikigai, our Technology & Innovation program is designed to equip students with essential tech skills from a young age.Our curriculum covers everything from basic coding and software use to advanced programming, robotics, and hardware design. By fostering problem-solving skills and encouraging hands-on projects.

Curriculum Overview

Elementary School

Basic Computer Skills

Learning to use basic Softawres like Paint and MS Suit, understanding operating systems, and basic file management.

Hardware Basics

Understanding basic computer components and their functions through interactive activities and demonstrations.

Introduction to Coding

Exploring simple coding concepts through visual programming tools and educational games.

Middle School

Intermediate Coding

Learning programming languages like Python or JavaScript and working on more complex coding projects.

Software Applications

Using tools for graphic design, video editing, and data management

Hardware Projects

Assembling and troubleshooting basic computer hardware components to understand their roles and functions.

High School

Advanced Programming

Mastering languages such as Java, C++, or HTML/CSS, and developing complex software applications or games.

Software Development

Creating and managing software projects, learning about software engineering practices and development methodologies.

Technology in Practice

Applying skills in internships, project-based learning, and tech fairs to gain real-world experience and build a professional portfolio.
